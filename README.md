# codebook_website

CodeBook is a modern e-learning platform designed for developers and tech enthusiasts to discover, explore, and purchase programming eBooks and courses. The application provides a seamless shopping experience with authentication, product management, and responsive design.

## Features

* Browse a collection of programming and software development eBooks
* View detailed information about each course or eBook
* User registration and authentication
* Secure login and logout functionality using JWT authentication
* Login as a guest without creating an account
* Add products to cart
* Update cart quantity and remove items from cart
* Persistent cart storage using browser storage
* Responsive design optimized for desktop, tablet, and mobile devices
* Product search and filtering capabilities
* Protected routes for authenticated users
